# Guess The Number

A simple console game written in Python.

## Description

The program generates a random number in a user-defined range.

The player tries to guess the number using hints:

* Above the number
* Below the number
* Close
* Very close

The game also counts the number of guesses.

## Features

* Random number generation
* Input validation
* Guess counter
* Distance hints
* Range validation

## How to Run

```bash
python main.py
```

## Technologies

* Python 3
* random module
